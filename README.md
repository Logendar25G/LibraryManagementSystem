# Library Management System

This is a command-line Library Management System built with TypeScript. It allows you to manage users, books, issue and return books, and keeps records persistently using JSON files.


# Features

1. Add and view users (students or staff)
2. Add and view books (categorized as Fiction or NonFiction)
3. Issue books to users
4. Return books and calculate fines
5. View all issued/return records
6. Deactivate users
7. Data is saved in JSON files for persistence

# Tech Stacks & Dependencies

1. TypeScript - for basic coding
2. Node.js - to execute the console application
3. Prompt-sync - to get input from the user

# How to run

1. Install dependencies: npm install
2. Start the CLI application:tsx src/LibraryManager.ts
